* isEmpty() 와 isPresent() *

- isEmpty()와 isPresent()는 Optional 클래스에서 제공하는 메소드들임.
- 두 메소드 모두 Optional 객체가 값을 가지고 있는지 확인하는데 사용되지만, 그 방식에는 약간 차이가 있음.

1. isPresent() : 
- 이 메소드는 Optional 객체가 값을 가지고 있으면 true를 반환하고, 그렇지 않으면 false를 반환함.
- 즉, Optional 객체가 non-empty 상태인지 확인하는데 사용됨.
- ex ) 
Optional<String> optional = Optional.of("Hello");
System.out.println(optional.isPresent());  // true

Optional<String> emptyOptional = Optional.empty();
System.out.println(emptyOptional.isPresent());  // false


2. isEmpty() :
-  이 메소드는 Java 11부터 제공되며,
 Optional 객체가 비어있으면 true를 반환하고, 그렇지 않으면 false를 반환함.
- 즉, Optional 객체가 empty 상태인지 확인하는데 사용됨.
- ex )
Optional<String> optional = Optional.of("Hello");
System.out.println(optional.isEmpty());  // false

Optional<String> emptyOptional = Optional.empty();
System.out.println(emptyOptional.isEmpty());  // true


=> isPresent()와 isEmpty()는 서로 반대되는 개념임.
 이 두 메소드를 적절하게 사용하면 Optional 객체의 상태를 정확하게 확인하고 처리할 수 있음.
