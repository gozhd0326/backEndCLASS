퍼블릭 키워드 ( public keyword )

- Java에서 접근 제어자(access modifier) 중 하나로, 해당 메서드나 변수가 어디에서나 접근 가능하다는 것을 나타냅니다.

- public으로 선언된 메서드나 변수는 다른 클래스, 패키지에서도 접근할 수 있습니다.

- 인터페이스에서 메서드를 선언할 때 public 키워드를 명시적으로 붙이지 않아도, 모든 메서드는 기본적으로 public입니다.
=>따라서 인터페이스 내에서 메서드 앞에 public을 생략해도 해당 메서드는 자동으로 public 접근 제어를 갖게 됩니다.

- public이 붙은 메서드와 없는 메서드 사이에 실질적인 차이는 없으며, 이는 단지 코드 스타일의 차이일 뿐입니다.

- 명시적으로 public을 붙이는 것은 코드의 가독성을 높이고,
 메서드가 인터페이스의 공개 API의 일부임을 명확히 하는 효과가 있습니다.

- ex ) getAllLectureByPaidLectures와 getAllLectureByFreeLectures 메서드에 public 키워드가 생략되었어도,
 이들은 public 메서드로 간주됩니다.
 그러나 일반적으로 인터페이스 내에서 메서드를 선언할 때는 public을 생략하는 것이 일반적인 관례입니다.
